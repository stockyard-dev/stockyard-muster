# Stockyard Muster

**On-call schedule and incident log.** Who's on call, what incidents happened, what was the resolution. PagerDuty is $19/user/mo. This is a SQLite file and a dashboard. Single binary, no external dependencies.

Part of the [Stockyard](https://stockyard.dev) suite of self-hosted developer tools.

## Free vs Pro

| Feature | Free | Pro ($4.99/mo) |
|---------|------|----------------|
| Members | 5 | Unlimited |
| Incidents | 20 | Unlimited |
| On-call schedule | ✓ | ✓ |
| Webhook alerts | — | ✓ |
